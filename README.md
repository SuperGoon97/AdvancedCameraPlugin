# AdvancedCameraPlugin
Advanced camera plugin for godot 4.5.1

########################################\
		INSTALLATION\
########################################

Drag and drop the addons folder into your res:// folder of your project

Goto project settings then go to plugins

enable the plugin 

reload the current project 

#######################################\
		HOW TO USE\
#######################################
